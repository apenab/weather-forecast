# Weather forecast
Example of using the Openweather API with nestjs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## open the following URL: http://localhost:3000/api/


## API endpoint examples:
```bash
GET /weather :> Returns the weather forecast based on the user's IP address
```
